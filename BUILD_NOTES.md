# Build Notes

## Android Studio

1. Repository in Android Studio oeffnen.
2. Gradle Sync ausfuehren.
3. Ein Android-Geraet oder einen Emulator auswaehlen.
4. App-Modul starten.

## Aktueller Prototyp

Die App ist bewusst einfach gehalten und nutzt native Android Views. Dadurch braucht sie keine Compose-Abhaengigkeiten und ist leichter als erster Build-Test.
